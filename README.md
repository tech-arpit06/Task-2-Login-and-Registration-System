# Task-2-Login-and-Registration-System
Login and Registration System

This project is a console-based Login and Registration System developed in C++. 
It allows users to create accounts and log in using their credentials. User data is stored in a text file using file handling concepts.

Features
User Registration
User Login Authentication
Duplicate Username Validation
File-Based Data Storage
Error and Success Messages
Menu-Driven Interface
Technologies Used
C++
File Handling
Functions
Conditional Statements
Loops
Working
Users register with a username and password.
The system checks for duplicate usernames.
Credentials are stored in a text file (users.txt).
Registered users can log in using their credentials.
The system verifies user information and displays appropriate messages.
Learning Outcomes
Understanding File Handling in C++
Implementing Authentication Systems
Working with Functions and Loops
Managing User Data
Input Validation Techniques
